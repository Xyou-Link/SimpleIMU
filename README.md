# 对此陀螺仪的简述



## 文件构成

- A_IMU为硬件工程
- 剩下文件为适配STM32主控的解算代码



## 软件说明事项

- IMU为应用层，LSM6DSRTR为底层

- 移植时只需修改LSM6DSRTR中的引脚定义，与IMU文件中的系统主频

- 此文件为一个项目上的兼容设计，如果使用额外的IMU想套用本解算只需更改IMUchoice结构体

  的函数指针与结构体指针的传入即可

## 解算说明

- 本解算采用方案为一阶互补滤波结合一维卡尔曼滤波级联
- 不采用四元数解算是因为四元数调参麻烦，对学弟学妹使用来说会比较麻烦，其实你观察我的工程我是做过四元数的解算的
- 只对XY做了卡尔曼滤波，如果你愿意，给Z加上一层也是可以的，在高响应下我习惯是不做
- 本解算是针对cortex-m4中有的FPU进行了一些优化，报错更换为数学函数即可

## 硬件说明
- 本人不是硬件专职，里面可能存在一些争议点
- scl与sda上配备上拉电阻是因为该陀螺仪同时兼容iic协议
- 陀螺仪为3V3 IN 可不焊接稳压芯片，短接5V与3V3即可
